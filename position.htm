итого:

static

иначе называется "без пазицинирования

realitive

сдвигает элемент относительно текущего места

  • противоположные границыleft/right(top/bottom)одновременно указать нельзя
  • окружающие элементы ведут себя так будто элемент никуда не сдвигался

absolute

визуально переносит элемент на новое место
НОВОЕ место вычесляется по координатам left/top/right/bottom относительно ближайшего позициорования родителя если такого родителя нет то им считается окно

  • ширина элемента по умолчанию устонавливается по содержимому
  • можно указать одновременно противоположные границы left/right(top/botom) элемент при этом растянется
  • окружающие элементы заполняют освободившееся место

fixed

подводит обсолютную позиционирования при коором элемент привязывается к координатам окна а не документа
