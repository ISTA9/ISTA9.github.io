Основные HTML теги

ТЕг Предзначения тега
<html></html> указывает программе просмотра страниц что внутри будет html документ
<head> </head> определяет место где помещается различная информация не отображаемая в теле документа.Здесь распологается тег название документа и прочие теги для настройки страницы
<body> </body> определяет видимую часть документа
